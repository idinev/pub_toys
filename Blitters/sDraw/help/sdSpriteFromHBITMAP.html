sdSpriteFromHBITMAP

sdSpriteFromHBITMAP creates an sSprite from the specified HBITMAP


sSprite* sdSpriteFromHBITMAP(HBITMAP hBitmap);

Parameters

hBitmap
a valid HBITMAP, loaded into GDI

Return Value

Returns a pointer to the new sSprite object. [ASM: modifies only EAX.]

Remarks

sdCreateBlankSprite fails only if it couldn't allocate wid*hei*4 bytes, or the specified hBitmap is not valid
The sSprite is marked with dwFlags=0 ("solid bitmap"), the presence of alpha-channel isn't automatically detected.
