sdGetPixel

sdGetPixel gets the color of the specified pixel


long sdGetPixel(long x,long y);

Parameters

x,y
coordinates of the pixel

Return Value

- NULL, if the x:y pixel isn't in the current clip-rectangle.
- The color of the pixel, masked with 0xFFFFFF (to remove garbage-alpha), if x:y belongs to the clip-rectangle.

Remarks

sdGetPixel takes up to 30 cycles to execute. If you need an even faster approach, you can use sdLockRect.

See Also

sdSetPixelA, sdSetPixel, sdLockRect
