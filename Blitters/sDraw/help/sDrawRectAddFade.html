sDrawRectAddFade

sDrawRectAddFade additive-blends a rectangle with the specified dwColor and the intensity of this dwColor (Alpha).


void sDrawRectAddFade(long x,long y,long wid,long hei,long dwColor,long Alpha);

Parameters

x,y,wid,hei
rectangle position and size
dwColor
color to use for drawing
Alpha
0..256 - intensity of the dwColor's light

Return Value

None. [ASM: All registers are preserved.]

Remarks

If you need a constant intensity of 256 (Alpha=256), you can also use sDrawRectROP(...,SD_ADDSAT);

See Also

sDrawRectAlpha, sDrawRectROP, sDrawRect
