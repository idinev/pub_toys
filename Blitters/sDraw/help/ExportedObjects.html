Exported objects (variables) from sDraw


long SD_TransparentColor; // transparent color, used in sBltTrans() and sBltTransAlpha(). By default it is magenta (0xFF00FF)

void* SD_CurFont; // pointer to the currently-selected font. Use it to change the font before calls to sTextOut(), sdPrintDec() and sTextOUTGetDrawnSize() .

Expert access only:

RECT SDBound; // current clipping-rectangle. Its position is absolute in the Client Area of the currently-drawn window. Adjusted by sdEnterClip()

POINT SDDrawOffs; // current X:Y offset for drawing operations. Again absolute in the Client Area of the currently-drawn window. Adjusted by sdEnterClip()

sSprite sTarget_Data; // data of the target-sprite. Get size and pointer to the bits from it. Filled-in by sdSetSourceSprite

sSprite sSource_Data; // data of the source-sprite. Get size and pointer to the bits from it. Filled-in by sdSetTargetSprite

HDC SDBackDC; // the GDI DeviceContext, that sDraw draws on (it's the back-buffer! ). When using GDI funcs on it, remember to adjust via SDDrawOffs !!
