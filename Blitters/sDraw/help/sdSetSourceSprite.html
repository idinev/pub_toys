sdSetSourceSprite

sdSetSourceSprite sets what sSprite to be used as source in subsequent sBlt*** calls.


void sdSetSourceSprite(sSprite* pSprite);

Parameters

pSprite
A loaded or created sSprite

Return Value

None. [ASM: All registers are preserved.]
