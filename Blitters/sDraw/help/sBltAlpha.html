sBltAlpha

sBltAlpha blends the source sSprite onto the backbuffer, with the specified Alpha-transparency (opacity)


void sBltAlpha(long x,long y,long wid,long hei,long x2,long y2,long Alpha);

Parameters

x,y,wid,hei,x2,y2
The usual parameters
Alpha
a value between 0 and 256. 0= invisible, 256=opaque

Return Value

None. [ASM: All registers are preserved.]

Example:

sBltAlpha(10,0,400,300,0,0,128);

Remarks

If Alpha>=256, sBltAlpha calls sBlt.
Premultiplied sSprites should not be used as the source! In such a case, the resulting image will appear darker than intended.

See Also
