sDrawRectROP

sDrawRectROP performs a ROP (Raster Operation) on a rectangle with the specified dwColor.


void sDrawRectROP(long x,long y,long wid,long hei,long dwColor,long dwROP);

Parameters

x,y,wid,hei
rectangle position and size
dwColor
color to use for drawing
dwROP
raster operation to use

Available ROPs

Assume dwOutColor is the destination pixel
SD_COPY
dwOutColor = dwColor
SD_XOR
dwOutColor ^= dwColor
SD_ADD
dwOutColor += dwColor
SD_SUB
dwOutColor -= dwColor
SD_OR
dwOutColor |= dwColor
SD_AND
dwOutColor &= dwColor
SD_SHR
dwOutColor >>= dwColor; (here dwColor=0..32)
SD_MUL
dwOutColor.Red = (dwOutColor.Red * dwColor.Red)>>8; (same for Blue and Green)
SD_ADDSAT
dwOutColor = Add_Saturate(dwOutColor,dwColor);
SD_SUBSAT
dwOutColor = Sub_Saturate(dwOutColor,dwColor);
SD_SHRSAT
dwOutColor.Red = dwOutColor.Red/2; (same for Blue and Green) (dwColor is not used). Makes the destination rectangle darker
SD_SHLSAT
dwOutColor = Add_Saturate(dwOutColor,dwOutColor); (dwColor is not used). Makes the destination rectangle brighter

Return Value

None. [ASM: All registers are preserved.]

Example:

sDrawRect(0,0,400,300,0xFFFFFF,SD_XOR);// invert the 400x300 rectangle at position 0:0
