sdSpriteFromILB

sdSpriteFromILB loads an sSprite from the memory-location specified


sSprite* sdSpriteFromILB(void* pSourceData);

Parameters

pSourceData
pointer to the start of the ILB's data

Return Value

Returns a pointer to the new sSprite object, or NULL if failed. [ASM: modifies only EAX.]

Example:

extern char embeddedImage[]; // remember to link this image as library
pAlphaImg1 = sdSpriteFromILB(embeddedImage);

Remarks

sdSpriteFromILBFile automatically marks if the sSprite has alpha-channel. If there's an alpha-channel, sdSpriteFromILBFile won't premultiply the alpha - you have to do it yourself, if necessary.
Usually you'd load ILB data from within the .exe file's .text/.code section.
