sBltTint

sBltTint performs an alpha-blend, by ignoring the color of the source-sSprite pixels, and using dwColor instead. sBltTint uses only the alpha-channel of the source!


void sBltTint(long x,long y,long wid,long hei,long x2,long y2,long dwColor);

Parameters

x,y,wid,hei,x2,y2
The usual parameters
dwColor
the color to draw with

Return Value

None. [ASM: All registers are preserved.]

Example:

sBltTint(0,0,400,300,0,0,0xFF0000); // tint to red

Remarks

If the source-sSprite doesn't have an alpha-channel, the resulting blend will not be visible (alpha defaults to 0).
