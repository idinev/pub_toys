sDrawRectFrame

sDrawRectFrame draws just the outline of the rectangle with the specified dwColor.


void sDrawRectFrame(long x,long y,long wid,long hei,long dwColor);

Parameters

x,y,wid,hei
rectangle position and size
dwColor
color to use for drawing

Return Value

None. [ASM: All registers are preserved.]

See Also

sDrawRect, sDrawRectFrame3D
