sTextOut

sTextOut draws the specified text with dwColor and the currently selected font (via the SD_CurFont variable)


sTextOut(long x,long y,char* lpszText,long dwColor)

Parameters

x,y
top-left corner to start drawing from
lpszText
the text to be drawn. Can be multi-line
dwColor
color to draw with

Return Value

None. [ASM: All registers are preserved.]

Example:

sTextOut(0,0,"some text\n new line",0xFFFFFF);

Remarks

Uses the font, pointed to by SD_CurFont.

See Also

sdLoadFont, sdPrintDec
