sBltTrans

sBltTrans performs a bit-block transfer, white treating pixels with a specific color as transparent. This color by default is magenta (#FF00FF). The SD_TransparentColor variable specifies it. sBltTrans doesn't care about the presense of alpha-channel for the source sSprite.


void sBltTrans(long x,long y,long wid,long hei,long x2,long y2);
extern long SD_TransparentColor;

Parameters

x,y,wid,hei,x2,y2
The usual parameters

Return Value

None. [ASM: All registers are preserved.]

Example:

sBltTrans(0,0,400,300,0,0);

Remarks

6-10 times faster than sBltAlpha, but its usage is limited. 20% slower than sBlt.
