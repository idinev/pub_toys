sdSetPixel

sdSetPixel sets the color of the specified pixel


void sdSetPixel(long x,long y,long dwColor);

Parameters

x,y
coordinates of the pixel
dwColor
color to set the pixel with

Return Value

None. [ASM: All registers are preserved.]

Remarks

sdSetPixel takes up to 30 cycles to execute. If you need an even faster approach, you can use sdLockRect

See Also

sdSetPixelA, sdGetPixel, sdLockRect
