sdStart

sdStart starts drawing on the current window's client area.


bool sdStart(HWND hWnd);

Parameters

hWnd
Handle of the window to start drawing on

Return Value

true - success
false - failed to start drawing, since the window is not visible. Must not continue drawing on it now.
[ASM: zero flag is set if failed]

Example:

void DrawWindow(HWND hWnd){
	if(!sdStart(hWnd))return; // must return on failure
	sDrawRect(0,0,2000,2000,0xFF0000); //fill with red
	sdEnd();
}

Remarks

Must end with sdEnd() if sdStart() is successful.
sDraw obtains the initial clipper via GDI's GetClipBox(), resets the draw-offset, clip-stack and source/target sSprites.
