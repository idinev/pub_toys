sDrawRect

sDrawRect draws a solid rectangle with the specified dwColor.


void sDrawRect(long x,long y,long wid,long hei,long dwColor);

Parameters

x,y,wid,hei
rectangle position and size
dwColor
color to use for drawing

Return Value

None. [ASM: All registers are preserved.]

Example:

sDrawRect(0,0,400,300,0xFFFFFF);// white 400x300 rectangle at position 0:0
