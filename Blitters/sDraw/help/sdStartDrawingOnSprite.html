sdStartDrawingOnSprite

Expert access only

sdStartDrawingOnSprite prepares the clipper and target-sprite for drawing on the whole specified sprite
sdEndDrawingOnSprite ends that operation, restores the previous state of the clipper.


void sdStartDrawingOnSprite(sSprite* pTargetSprite);
void sdEndDrawingOnSprite();

Parameters

pTargetSprite
sprite, on which to draw from now-on. The clipper is adjusted to start drawing at 0:0, and to match the size of the sprite. 

Return Value

All registers are preserved.

Example:

	sdStartDrawingOnSprite(pSprite1);
		sDrawRect(7,7,200,200,0xFF0000); // red
	sdEndDrawingOnSprite();

Remarks

You can do internal clipping with sdEnterClip() and sdLeaveClip(), and even nest it.
You cannot nest calls of sdStartDrawingOnSprite! Not that you'd ever need that...
