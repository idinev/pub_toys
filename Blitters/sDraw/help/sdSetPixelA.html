sdSetPixelA

sdSetPixelA blends the specified pixel with dwColor and Alpha


void sdSetPixelA(long x,long y,long dwColor,long Alpha);

Parameters

x,y
coordinates of the pixel
dwColor
color to set the pixel with
Alpha
opacity of dwColor

Return Value

None. [ASM: All registers are preserved.]

Remarks

sdSetPixelA takes up to 60 cycles to execute. If you need an even faster approach, you can use sdLockRect

See Also

sdSetPixelA, sdGetPixel, sdLockRect
