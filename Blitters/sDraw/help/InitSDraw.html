InitSDraw

InitSDraw initializes the sDraw library. Doesn't require the presence of any already-created window in your app.


InitSDraw(long wid,long hei);

Parameters

wid,hei
width and height of the back-buffer for sDraw. To save RAM, use the tightest possible size. Usually, wid=0 and hei=0 - sDraw will replace them with the screen-size.

Return Value

None. [ASM: All registers are preserved.]

Example:

InitSDraw(0,0);

Remarks

Should be called only once. Before app-exit, you should call FreeSDraw();
