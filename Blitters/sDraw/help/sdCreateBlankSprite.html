sdCreateBlankSprite

sdCreateBlankSprite creates an sSprite with the specified size


sSprite* sdCreateBlankSprite(long wid,long hei);

Parameters

wid,hei
dimensions of the new sSprite

Return Value

Returns a pointer to the new sSprite object. [ASM: modifies only EAX.]

Example:

pBackSprite = sdCreateBlankSprite(640,480);

Remarks

sdCreateBlankSprite fails only if it couldn't allocate wid*hei*4 bytes.
By default, the new sSprite has dwFlags=0, meaning that its alpha-channel won't be used in most sBlt*** operations (i.e the sprite is "solid").
