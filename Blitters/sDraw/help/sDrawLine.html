sDrawLine

sDrawLine draws a line from x:y to x2:y2 with subpixel accuracy.


void sDrawLine(long x,long y,long x2,long y2,long dwColor)

void sDrawLineH(long x,long y,long width,long dwColor)
void sDrawLineV(long x,long y,long height,long dwColor)

void sDrawLineH_HalfAlpha(long x,long y,long width,long dwColor)
void sDrawLineV_HalfAlpha(long x,long y,long height,long dwColor)

Parameters

x,y
coordinates of starting point
x2,y2
coordinates of ending point (Note: order of points doesn't matter)
dwColor
color of the line


width/height - on sDrawLineH and sDrawLineV, the length of the line

Return Value

None. [ASM: All registers are preserved.]

Example:

sDrawLine(10,10,1000,30,0xFF0000);

Remarks

Internally, sDrawLine uses sdSetPixelA(), doesn't do internal clipping of the line beforehand. Thus, do not expect max performance.

sDrawLineH and sDrawLineV are optimized versions to draw respectively horizontal and vertical lines.

sDrawLineH_HalfAlpha and sDrawLineV_HalfAlpha are optimized versions to draw respectively horizontal and vertical lines, with opacity = 50%
