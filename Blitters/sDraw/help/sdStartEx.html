sdStartEx

sdStartEx starts drawing on the current window's client area.


bool sdStartEx(HWND hWnd,long dwType);

Parameters

hWnd
Handle of the window to start drawing on
dwType
One of three values:
  • a) SDSTARTEX_ONUPDATEREGION - draws only on the update-region rectangle.
  • b) SDSTARTEX_ONDRAWREGION - draws to the whole window (default behaviour of sdStart() )
  • c) pointer to a RECT structure - draws only to the specified rectangle of the window.

    • Return Value

    true - success
    false - failed to start drawing, since the window is not visible. Must not continue drawing on it now.
    [ASM: zero flag is set if failed]


    Remarks

    Must end with sdEnd() if sdStartEx() is successful.
