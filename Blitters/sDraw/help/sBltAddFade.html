sBltAddFade

sBltAddFade performs additive-blending transfer, while attenuating the source intensity.


void sBltAddFade(long x,long y,long wid,long hei,long x2,long y2,long Fader);

Parameters

x,y,wid,hei,x2,y2
The usual parameters
Fader
0..256 : 0=no light, 256=full-intensity

Return Value

None. [ASM: All registers are preserved.]

Example:

sBltAddFade(0,0,400,300,0,0,161);

Remarks

3-4 times slower than sBltAdd.

See Also

sBltAddFadeParam, sBltAdd, sBlt, sdSetSourceSprite
