sBltTransAlpha

sBltTrans performs an alpha-blend transfer, white treating pixels with a specific color as transparent. This color by default is magenta (#FF00FF). The SD_TransparentColor variable specifies it. sBltTransAlpha doesn't care about the presense of alpha-channel for the source sSprite.


void sBltTransAlpha(long x,long y,long wid,long hei,long x2,long y2,long Alpha);
extern long SD_TransparentColor;

Parameters

x,y,wid,hei,x2,y2
The usual parameters
Alpha
0..256, specifies opacity

Return Value

None. [ASM: All registers are preserved.]

Example:

sBltTransAlpha(0,0,400,300,0,0,126);
