sdSpriteFromILBFile

sdSpriteFromILBFile creates an sSprite from the specified ".ilb" file.


sSprite* sdSpriteFromILBFile(char* lpszFileName);

Parameters

lpszFileName
pointer to a string, containing the file name of an ".ilb" file.

Return Value

Returns a pointer to the new sSprite object, or NULL if failed. [ASM: modifies only EAX.]

Example:

pAlphaImg1 = sdSpriteFromILBFile("AlphaImage1.ilb");

Remarks

sdSpriteFromILBFile automatically marks if the sSprite has alpha-channel. If there's an alpha-channel, sdSpriteFromILBFile won't premultiply the alpha - you have to do it yourself, if necessary.
