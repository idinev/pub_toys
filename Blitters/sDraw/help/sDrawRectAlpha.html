sDrawRectAlpha

sDrawRectAlpha draws a transparent rectangle with the specified dwColor and opacity (Alpha).


void sDrawRectAlpha(long x,long y,long wid,long hei,long dwColor,long Alpha);

Parameters

x,y,wid,hei
rectangle position and size
dwColor
color to use for drawing
Alpha
0..256. opacity of drawing

Return Value

None. [ASM: All registers are preserved.]

Example:

sDrawRectAlpha(0,0,400,300,0xFFFFFF,68);// white 400x300 rectangle at position 0:0, opacity 68

Remarks

if Alpha=128, the blend is 4 times faster.
