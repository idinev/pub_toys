sdPrintDec

sdPrintDec draws the specified signed number with dwColor and the currently selected font (via the SD_CurFont variable)


sdPrintDec(long x,long y,long nValue,long dwColor)

Parameters

x,y
top-left corner to start drawing from
nValue
signed integer
dwColor
color to draw with

Return Value

None. [ASM: All registers are preserved.]

Example:

sdPrintDec(0,0, 177,0xFFFFFF);

Remarks

Internally calls sTextOut()

See Also

sdLoadFont, sTextOut
