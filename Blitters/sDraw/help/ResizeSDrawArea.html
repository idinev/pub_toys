ResizeSDrawArea

ResizeSDrawArea resizes the back-buffer of sDraw. Usually called when the screen-size has been changed (after WM_DISPLAYCHANGE)


void ResizeSDrawArea(long wid,long hei);

Parameters

wid,hei
new dimensions for the back-buffer. If wid=0 and hei=0, the full-screen size is used

Return Value

None. [ASM: All registers are preserved.]

Example:

case WM_DISPLAYCHANGE:
     ResizeSDrawArea(0,0);
     break;

Remarks

If the screen-size hasn't actually changed, ResizeSDrawArea() will return immediately.

See Also

InitSDraw, FreeSDraw, sdStart
