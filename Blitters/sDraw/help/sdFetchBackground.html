sdFetchBackground

Intermeddiate access only

sdFetchBackground is used to copy a snapshot of the window's GUI into the back-buffer of sDraw (which is used to draw on).  This is useful for splash-screens, to produce a nice effect with alpha-channel.


void sdFetchBackground();

Return Value

None. [ASM: All registers are preserved.]

Example:

void DrawWindow(HWND hWnd){
	if(!sdStart(hWnd))return;
	sdFetchBackground(); // usually use it after sdStart()
	// now draw some stuff over
	....
	sdEnd();
}
