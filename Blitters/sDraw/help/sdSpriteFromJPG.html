sdSpriteFromJPG

sdSpriteFromJPG loads an sSprite from the memory-location specified, which should contain the data of a JPG/PNG/GIF image.


sSprite* sdSpriteFromJPG(void* pSourceData,long DataSize);

Parameters

pSourceData
pointer to the start of the JPG image's data
DataSize
size, in bytes, of the image data

Return Value

Returns a pointer to the new sSprite object, or NULL if failed. [ASM: modifies only EAX.]


Remarks

GIF and PNG images cannot have their alpha-transparency correctly-loaded.
You must have called "CoInitialize(0)" at least once before using sdSpriteFromJPG
