sdSpriteFromBitmapFile

sdSpriteFromBitmapFile creates an sSprite from the specified .bmp file


sSprite* sdSpriteFromBitmapFile(char* lpszFileName);

Parameters

lpszFileName
pointer to a string, containing the file name of a ".bmp" file.

Return Value

Returns a pointer to the new sSprite object, or NULL if failed. [ASM: modifies only EAX.]

Example:

pImg1 = sdSpriteFromBitmapFile("image1.bmp");

Remarks

sdSpriteFromBitmapFile cannot load ".jpg" , ".gif" or ".png". Internally, it uses the LoadImage() API and sdSpriteFromHBITMAP().
