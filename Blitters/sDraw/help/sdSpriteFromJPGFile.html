sdSpriteFromJPGFile

sdSpriteFromJPGFile creates an sSprite from the specified .jpg/.png/.gif file.  Useful if you want full-color images. (ILB files have 16-bit color)
You must have called "CoInitialize(0)" at least once before using sdSpriteFromJPG


sSprite* sdSpriteFromJPGFile(char* lpszFileName);

Parameters

lpszFileName
pointer to a string, containing the file name of an ".jpg" file.

Return Value

Returns a pointer to the new sSprite object, or NULL if failed. [ASM: modifies only EAX.]

Example:

pAlphaImg1 = sdSpriteFromJPGFile("Smily.jpg");
