sDrawRectFrame3D

sDrawRectFrame3D draws a solid rectangle with the specified dwColor, draws the top and left borders with dwLightColor, and draws the bottom and right borders with dwDarkColor.


void sDrawRectFrame3D(long x,long y,long wid,long hei,long dwColor,long dwLightColor,long dwDarkColor);

Parameters

x,y,wid,hei
rectangle position and size
dwColor
color to use for filling
dwLightColor
color to use for drawing the top and left borders. Usually brighter than dwColor
dwDarkColor
color to use for drawing the bottom and right borders. Usually darker than dwColor

Return Value

None. [ASM: All registers are preserved.]

Example:

sDrawRectFrame3D(0,0,80,128,RGB(128,160,180),RGB(152,192,218),RGB(96,122,138));
