ILB files/data

What are ILB files?

An .ILB file (introduced in sDraw) is a compressed .bmp (without alpha-channel) or .tga (with alpha-channel). The compression is a bit lossy, since it initially crunches RGB888 (24-bit color) into RGB565 (16-bit color). The alpha channel stays intact (8-bit), if present.
Currently in sDraw, ILB files are mostly useful for images that have alpha-channel.


Making .ilb files/libs

• To a file: Drag-drop a .bmp or .tga file over PackToILB.exe
• To a .lib: Drag-drop a .bmp or .tga file over PackBitmap.exe. You need the MASM32 package, otherwise PackBitmap will crash.
.bmp files must be 24bpp
.tga files must be 32bpp without RLE compression, origin at top-left.
